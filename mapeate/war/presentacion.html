Descripción General

Controla donde has viajado y donde quieres viajar desde un cómodo board en el que podrás tener el control del mundo.
También puedes descubrir los lugares recorridos por tus amigos y promover sugerencias y opciones de viajes.
Con esta web podrás tener el control de tu mundo



Casos de Uso

En principio la idea es tener una parte pública(sin necesidad de registro)
en la que el usuario podrá visualizar un mapa de ejemplo y probar el uso básico
de la web. Pero para tener un control pleno se necesitará registrar.
Una vez registrado, el usuario podrá a través de un intuitivo mapamundi, situar
sus chinchetas en los países que ha visitado o que pretende de visitar. También
podrá visualizar comentarios de otros usuarios sobre esos países y de esta forma
saber que sitios son preferibles en la visita.
El usuario también puede poner sus propios comentarios sobre lugares visitados.
De esta forma se controla de una forma rápida e intutitiva todas las posibilidades de viajar.


Tenologías a usar

La idea no está clara, pero en principio para esta web se usará PHP y J2EE . Se desarrollará en XHTML para usar validadores.
Para el estilo se usará CSS, y el uso de ciertas herramientas que faciliten el
entendimiento y la accesibilidad a la web. Para ello se está usando el APP Engine de Google para el uso de Servlets. Se usa la librería Quercus que da soporte para PHP. También se usará JavaScript y varios efectos de JQuery.
